# avalanchesass_function_map_get_deep
Get values from deep nested maps.

## Install
```
npm install avalanchesass_function_map_get_deep --save
```

## Basic usage
```css
/* Import the main file. */
@import 'avalanchesass_function_map_get_deep';
```
